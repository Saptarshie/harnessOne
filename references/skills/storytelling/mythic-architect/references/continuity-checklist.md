# Continuity Checklist — Per-Chapter Verification

Run BEFORE critic dispatch and AFTER surgical revision. Subagents (writer/critic)
operate in isolated contexts and will drift from prior-chapter canon. Verify:

## Character Details

### Scar Locations (example: Nera Castell, THE CHRYSALIS PROTOCOL)
- [ ] Ch.01: scar on **right forearm**, pale line, 6 cm
- [ ] Ch.02: scar on **right forearm** (continuity fix applied — was "left collarbone")
- [ ] Ch.03: scar on **right forearm** (continuity fix applied — was "left forearm" + inconsistent backstory)
- [ ] All subsequent: verify location matches canon, verify backstory doesn't shift

### Data Provenance (example: Theron's crystal, THE CHRYSALIS PROTOCOL)
- [ ] Ch.02: Theron copied **origin data** (Calibrator's artificial nature) from deep archives
- [ ] Ch.03: Theron's data-strand contains **origin data**, not Directorate telemetry (was incorrectly written as "stolen from Optimization Directorate")
- [ ] All subsequent: verify data type matches Ch.02, verify acquisition method isn't retroactively changed

### General Continuity
- [ ] Age of each POV character consistent across chapters
- [ ] Faction affiliation unchanged
- [ ] Location of key objects (artifact, crystal, falsified log) tracks correctly
- [ ] Timeline: hours/days since Calibrator's death match across POVs
- [ ] Somatic tells don't mutate (if character taps fingers under stress, they shouldn't hum instead in the next chapter without valid in-story reason)

## Common Drift Patterns (from real sessions)

| Pattern | Example | Root Cause |
|---------|---------|-----------|
| **Scar migration** | Scar moves from right forearm → left forearm → collarbone | Subagent doesn't have Ch.01 canon; invents plausible-but-wrong detail |
| **Data source shift** | "Stole from Directorate six hours before" vs "copied origin data during Consensus" | Subagent conflates character's knowledge with plot knowledge the character doesn't have |
| **Backstory inflation** | "Since childhood, no medic could explain" vs "adult wound from a lover" | Subagent adds detail to "enrich" the character; detail contradicts canon |
| **Temporal drift** | "Third hour" in epigraph vs "three hours old" in prose vs actual timeline | Epigraph vs inline timekeeping fall out of sync |

## Fix Protocol

When a continuity error is found:
1. Identify which chapter established the correct canon (usually the FIRST appearance)
2. Patch the drifting chapter to match the original
3. Do NOT "split the difference" — original canon wins
4. Add the specific detail to this checklist so future chapters can be verified against it
